基本操作
---
- 新增文件：git add {readme.txt}
- 提交并注释：git commit -m "{wrote a readme file}"
- 仓库初始化：git init
- 取得仓库状态：git status

分支管理
---
**参考指针或链表模型**

-新建分支并切换：git checkout [-b] {dev}
-查看分支：git branch
-切换分支：git checkout [master]
-合并分支：git merge
-删除分支：git branch -d <name>
